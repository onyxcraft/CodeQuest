# CodeQuest - Algorithm Puzzle

CodeQuest teaches algorithms through puzzle gameplay. Built with Swift, SwiftUI, and SpriteKit.

## Overview

CodeQuest is a puzzle game where you program a robot to navigate mazes by creating command sequences. Each level introduces different algorithmic concepts, from basic movement to loops and conditionals.

## Features

- **5 Progressive Levels**: From basic navigation to complex pathfinding
- **Visual Programming**: Drag and drop commands to build your program
- **Real-time Execution**: Watch your robot execute commands step-by-step
- **Algorithm Concepts**: Learn loops, conditionals, and nested structures
- **Clean UI**: Dark theme with pixel-inspired design

## Levels

1. **Hello World** - Basic movement and turning
2. **Loop It** - Introduction to repeat loops
3. **Branch Out** - Conditional statements with if-wall
4. **Nested** - Combining loops and conditionals
5. **Pathfinder** - Complex maze solving

## Tech Stack

- Swift 5.0
- SwiftUI for UI
- SpriteKit for game canvas
- MVVM architecture
- macOS 14.0+

## Building

```bash
xcodebuild -project CodeQuest.xcodeproj -scheme CodeQuest -configuration Release build
```
